
# ¿Qué es una Pull Request?

Una **Pull Request (PR)** es una solicitud para fusionar cambios de una branch a otra. 

## Ventajas:
- Revisión de código por equipo
- Discusión y sugerencias
- Historial de cambios documentado

## Desventajas:
- Proceso más lento
- Requiere aprobación
